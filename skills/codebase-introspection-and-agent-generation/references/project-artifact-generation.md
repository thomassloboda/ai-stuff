# Project Artifact Generation Reference

## Goal

Turn repository analysis into a small, high-value set of project-specific AI guidance artifacts.

## Preferred artifact set

Generate or recommend:
- `STACK_ANALYSIS.md`
- `PROJECT_CONVENTIONS.md`
- `AGENTS.md`
- `WORKFLOWS.md`
- `RECOMMENDED_SKILLS_AND_AGENTS.md`

## Rules

- Prefer concise, useful artifacts over exhaustive documentation.
- Generate only what clearly improves developer speed or agent reliability.
- Do not create project-specific agents or skills unless the repository truly needs specialized behavior.
